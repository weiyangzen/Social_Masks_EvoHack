# A2A 社交面具蓝图

## 目标
- 先建立关系型“社交面具清单”骨架。
- 每个勾选项代表一个待定义的面具原型，不是最终成稿。
- `[ ]` 表示尚未完成，后续可继续补全角色定义、触发场景、话术风格、边界与风险。

## 执行约束
- 权威执行清单仅包含以下 9 个关系章节中的 `[ ]` / `[x]` 条目。
- 每个勾选项的完成条件是：已经生成对应的 `MasksSkills/**/iNN.md` skill 文件，且文件结构满足统一模板。
- `Docs/todos_YYYYMMDD.md` 只是每日执行镜像，不是第二需求源。
- 统一 skill 模板与验收字段见 `MasksSkills/README.md`。

## 执行分片
- worker-1：家人 + 同事
- worker-2：跑腿 + 嘴替
- worker-3：搭子 + 两性
- worker-4：求职招聘 + 投融资
- worker-5：闲置

## 后续补完字段
- 面具名称

- 典型场景
- 用户输入
- 输出风格

- 风险提示
- 示例对话
